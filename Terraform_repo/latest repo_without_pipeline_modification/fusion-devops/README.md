# Introduction 
Configuration Files for all FieldSmart components are stored in this repository

# Getting Started

Configuration Precedence:

Tenant -> Application -> Environment

Common configuration values of tenant would be overriden by common application specific values.
Common Application Specific values will be overriden by application's environment specific values.
